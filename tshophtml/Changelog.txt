**

##  New Release: Version 1.5 ##

**


TSHOP is a modern responsive HTML5 CSS3 ecommerce website template made with Bootstrap v3.3.5 Compatible with all modern browsers and usable with any device.

## Features##

 -  Fully Responsive 
 -  Bootstrap v3.3.5
 -  Valid HTML5 Markup 
 -  10+ Colors Schemes and easy to set with skin.less
 -  60+ HTML Pages 
 -  5+ different slider variations
 -  3 different header variations
 -  5-step checkout Page
 -  One Page checkout 
 -  5 types of product details variations with lots of option. 
 -  Product zoom + Litebox + Ajax Product Modal.
 -  Customs design jQuery cycle & Swiper Slider.
 -  3 Type of About page
 -  2 Type of Category page
 -  Parallax Story Page
 -  Blogs
 -  Contact with Google Map
 -  CSS3 Animations
 -  Detailed Documentation
 -  Cross browser 
 -  2000+ SATISFIED CUSTOMERS
 -  Available support
 -  Free lifetime updates
 -  and more...
